📋 設定指南

步驟 1：建立 Google Sheets

  1. 建立新的 Google 試算表
  2. 將第一個工作表命名為 "bookings"
  3. 在第一列加入欄位標題：timestamp, name, phone, email, service, date, time, notes, status

步驟 2：建立 Google Apps Script Web App

  1. 在 Google Sheets 中，點選「擴充功能」→「Apps Script」
  2. 貼上提供的 Apps Script 程式碼
  3. 部署為 Web App，設定存取權限為「任何人」
  4. 複製 Web App URL

步驟 3：取得 Google Sheets API Key

  1. 前往 Google Cloud Console
  2. 建立新專案或選擇現有專案
  3. 啟用 Google Sheets API
  4. 建立 API 金鑰
  5. 設定 API 金鑰限制（建議限制來源網域）

步驟 4：更新程式碼

  1. 將 YOUR_API_KEY 替換為您的 API 金鑰
  2. 將 YOUR_SPREADSHEET_ID 替換為試算表 ID
  3. 將 YOUR_GOOGLE_APPS_SCRIPT_URL 替換為 Web App URL

步驟 5：部署到 GitHub Pages

  1. 將此 HTML 檔案上傳到 GitHub repository
  2. 在 Settings → Pages 中啟用 GitHub Pages
  3. 選擇來源分支和資料夾
  4. 等待部署完成，即可使用您的預約系統
